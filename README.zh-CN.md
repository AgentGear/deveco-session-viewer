# DevEco Code Session Viewer

[English](./README.md) | 中文

一个用于 deveco/opencode 的插件，提供本地 Web UI 来浏览、搜索和管理对话会话，支持主题切换和键盘导航。

## 功能特性

### 会话管理
- **会话列表视图** - 浏览所有对话会话，支持搜索、排序和筛选
- **会话详情视图** - 查看完整的对话历史，支持 Markdown 渲染
- **重命名会话** - 直接在列表中编辑会话标题（与 deveco TUI 同步）
- **删除会话** - 通过确认对话框永久删除会话
- **收藏功能** - 为会话添加星标，将其固定在列表顶部（存储在浏览器 localStorage 中）
- **导出功能** - 将会话导出为 Markdown 或 JSON，内容遵循 Clean Mode 设置
- **复制消息** - 将单条消息内容复制到剪贴板

### 搜索与筛选
- **关键词搜索** - 在会话列表中按关键词筛选会话
- **会话内搜索** - 使用 Ctrl+F 在对话中搜索，支持匹配高亮和导航
- **排序** - 按时间、费用或 Token 数排序会话（升序/降序）
- **模型筛选** - 多选复选框筛选器，按特定模型显示会话
- **收藏筛选** - 切换显示仅收藏的会话
- **时间分组** - 按时间排序时，会话按 今天 / 昨天 / 本周 / 本月 / 更早 分组

### 统计仪表板
- **标签页导航** - 在会话和统计视图之间切换
- **时间范围选择器** - 按今天、7 天、30 天或全部时间筛选统计数据
- **摘要卡片** - 总费用、总 Token 数、会话数、平均每次会话费用
- **图表**（由 Chart.js 驱动）：
  - 每日费用趋势（折线图）
  - 按模型分布的费用（环形图）
  - 每日会话数（柱状图）

### 用户体验
- **实时更新** - 当会话或消息变化时，通过 Server-Sent Events (SSE) 自动刷新
- **Clean Mode** - 隐藏中间代理步骤，仅显示每轮的用户输入和最终助手响应
- **粘性头部** - 会话标题、搜索栏和导航控件在滚动时保持可见
- **服务器信息** - 状态栏显示连接的项目目录和服务器 IP:端口
- **主题切换** - 在深色和浅色主题之间切换（偏好设置保存到 localStorage）
- **键盘导航** - 使用方向键导航会话，Enter 打开，Esc 返回

## 安装

### 1. 克隆仓库

```bash
git clone https://github.com/AgentGear/deveco-session-viewer.git
cd deveco-session-viewer
```

### 2. 安装依赖

```bash
npm install
```

### 3. 在 deveco 项目中配置

将插件添加到你的 `deveco.jsonc`（或 `opencode.jsonc`）：

```jsonc
{
  "plugin": [
    ["./path/to/deveco-session-viewer", { "port": 9876 }]
  ]
}
```

**选项：**
- `port`（可选）：HTTP 服务器端口，默认为 `9876`

### 4. 启动 deveco

```bash
bun dev
```

### 5. 打开 Web UI

在浏览器中访问 `http://localhost:9876`。

## 系统要求

- **deveco**: >= 1.15.0（或任何兼容的分支）
- **Node.js**: >= 18

## 使用说明

### 会话列表

- 浏览所有会话，按时间分组（今天 / 昨天 / 本周 / 本月 / 更早）
- 使用搜索框按关键词筛选会话
- 使用排序下拉菜单按时间、费用或 Token 数排序（再次点击切换方向）
- 使用模型筛选下拉菜单仅显示使用特定模型的会话（多选）
- 点击任何会话卡片上的星标图标进行收藏（收藏的会话固定在顶部）
- 点击铅笔图标重命名会话
- 点击垃圾桶图标删除会话（带确认）
- 点击任何会话卡片查看详情

### 会话详情

- **搜索** - 始终可见的搜索栏，用于在对话中查找文本（Enter/Shift+Enter 导航匹配项）
- **导出** - 将会话导出为 Markdown 或 JSON（遵循 Clean Mode）
- **Clean Mode** - 隐藏中间代理步骤，仅显示每轮的用户输入和最终助手响应
- **复制** - 悬停在任何消息上以复制其文本内容
- **返回** 按钮 - 返回会话列表
- 会话元数据：标题、创建时间、最后更新时间、模型、Token 使用量、费用

### 统计

- 切换到统计标签页查看使用分析
- 选择时间范围：今天、7 天、30 天或全部
- 查看摘要卡片和交互式图表

### 实时更新

Web UI 会在以下情况自动更新：
- 创建新会话
- 会话标题更改
- 会话被删除
- 当前查看的会话中添加新消息

更新通过 Server-Sent Events (SSE) 传递，断开连接时自动重连。

### 键盘快捷键

**会话列表：**
- `↑` / `↓` - 在会话之间导航
- `Enter` - 打开选中的会话
- `Esc` - 清除搜索（如果搜索框获得焦点）

**会话详情：**
- `Ctrl+F` / `Cmd+F` - 在对话中搜索
- `Enter` - 下一个搜索匹配项
- `Shift+Enter` - 上一个搜索匹配项
- `Esc` - 关闭搜索 / 返回会话列表

**通用：**
- 点击标题栏中的主题切换按钮（☀️/🌙）在浅色和深色主题之间切换

## 架构

此插件在运行时**零依赖**：

- 使用 Node.js 内置的 `node:http` 和 `node:os` 模块
- 将完整的 Web UI 作为字符串嵌入（无静态文件）
- 通过进程内客户端与 deveco 通信（无网络开销）
- 类型定义内联（无外部 `@opencode-ai/plugin` 依赖）
- Chart.js 通过 CDN 加载用于统计图表
- 主题偏好和收藏存储在浏览器 localStorage 中

插件启动一个 HTTP 服务器：
1. 在 `/` 提供嵌入的 Web UI
2. 提供会话数据的 REST API 端点
3. 在 `/api/events` 通过 SSE 流式传输实时事件

## API 端点

| 端点 | 方法 | 描述 |
|---|---|---|
| `/` | GET | Web UI |
| `/api/info` | GET | 项目信息（directory, projectId, serverIP, serverPort） |
| `/api/sessions?search=...&limit=...` | GET | 列出会话，支持可选的搜索和限制 |
| `/api/sessions/:id` | GET | 获取会话详情 |
| `/api/sessions/:id` | PATCH | 更新会话（标题） |
| `/api/sessions/:id` | DELETE | 删除会话 |
| `/api/sessions/:id/messages?limit=...` | GET | 获取会话消息 |
| `/api/events` | GET | SSE 流，用于实时更新 |
